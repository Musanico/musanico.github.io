几个基本语句

JS编写的位置

1.放在HTML文件的开头处,用script标签划分区域(不常用)

2.引入外部链接(常用)

地址一定要使用相对地址,不要用绝对地址

3.写在标签中(不常用)

注释

标识符

JS中可以用中文作为标识符,但是不会用

数据类型

在JS中有六种数据类型:
1.String 字符串
2.Number 数字
3.Boolean 布尔(逻辑)
4.Null 空值
5.Undefined 未定义
6.Object 对象

除了Object属于引用数据类型,其他都是基本数据类型

两个小知识点

输出UNICode编码

1.JS

2.HTML

代码块

JS中用{}来给语句进行分组,同一个{}中的语句被称为一组语句(代码块),要么都执行要么都不执行

关系运算符知识点

1,任何值和NaN进行比较的结果都是false
2.当两边都是字符串时,会直接按顺序比较两边字符串的UNICode编码,如果一样的话,则下一位进行比较
3.比较两个字符串类型的数字时,一定要先转型再进行比较

进制输入

在JS中,如果要表示16进制的数字,要以0x开头
8进制以0开头
2进制以0b开头,但不是所有浏览器都支持
他们的输出结果都是以10进制显示的\

强制类型转换

1.数字转字符串

2.字符串转数字

若为true,则转成1
若为false,则转成0
空字符串和null都为0
带有字符的字符串和undefined都为NaN

3.从字符串中读取数字

条件:第一位一定要为数字

4.从字符串读取浮点数

5.转成Boolean类型
数字除了0和NaN,其余都为true
字符串除了空字符串(字符串不含任何东西,包括空格).其余都为true
null和Undefined都为false
对象也会转换为true

一元运算符

1.”+”+数字字符串可以快速将字符串转为数字
2.”-“ 对于非number的值,会先转换成number再取反

条件运算符

条件运算符也叫三元运算符
语法:
条件表达式?语句1:语句2;    
执行流程:先执行条件表达式,若为true,执行语句1,false执行语句2

下面这个一般不用,不方便阅读

对象

对象的分类:
1.内建对象:由ES标准中定义的对象,在任何ES的实现中都可以使用,比如Math String Number Boolean Function Object
2.宿主对象:由JS得运行环境提供的对象,目前来讲主要指由浏览器提供的对象,比如 BOM DOM console document
3.自定义对象:由开发人员自己创建的对象
